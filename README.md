# Visor (Tauri + React + TS)

Minimal Phase 1 skeleton: codemap canvas on top, pinned-context sidebar on the left, terminal docked at the bottom.

## Prerequisites
- Node 18+ (for Vite dev server)
- Rust toolchain + `cargo` (for Tauri backend)
- `npm` (using the repo-committed lockfile)

## Install
```bash
npm install
```

## Run (dev)
```bash
npm run tauri dev
```

## Build (frontend only)
```bash
npm run build
```
